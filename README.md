# Pokemon: Maroon And Gold for macOS
Pokemon Maroon and Gold. A Pokemon-style mini game for macOS.<br>
<br>
<h3>GOAL</h3>
Find Memorial Union as fast as possible!<br>

<h3>INSTRUCTION</h3>
Press RIGHT key on the Title screen to start the game.<br>
